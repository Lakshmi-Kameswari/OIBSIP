# 🧮 BMI Calculator (Python GUI)

## 📌 Project Overview

This project is a **GUI-based BMI (Body Mass Index) Calculator** built using **Python and Tkinter**.
The application allows users to enter their **weight and height**, calculate BMI instantly, and view the **corresponding health category**.

The interface is designed with a **modern dark theme and interactive buttons** for a better user experience.

---

## 🚀 Features

* Simple and user-friendly GUI
* Instant BMI calculation
* Automatic BMI category detection
* Input validation for incorrect values
* Reset button to clear inputs
* Modern dark theme interface
* Button hover effects for better interaction

---

## 🛠 Technologies Used

* **Python**
* **Tkinter** (GUI Library)

---

## ▶️ How to Run the Project

1. Make sure **Python 3.x** is installed.
2. Download or clone the repository.
3. Open terminal in the project folder.
4. Run the program:

```
python main.py
```

The BMI Calculator window will open.

---

## 📊 BMI Categories Used

| BMI Range      | Category      |
| -------------- | ------------- |
| Less than 18.5 | Underweight   |
| 18.5 – 24.9    | Normal Weight |
| 25 – 29.9      | Overweight    |
| 30 and above   | Obese         |
