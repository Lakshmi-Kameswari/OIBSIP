## 📌 Projects Included

### 1️⃣ Chat Application (Client–Server)

A real-time text-based chat application built using **Python socket programming** and **Tkinter GUI**.
It allows multiple users to communicate in real-time using a **client-server architecture**.

**Features:**

* Real-time messaging between users
* Client–server communication using sockets
* GUI interface built with Tkinter
* Nickname-based user identification
* Message timestamps
* Multiple users can connect simultaneously

**Technologies Used:**

* Python
* Socket Programming
* Tkinter (GUI)
* Threading

---

### 2️⃣ BMI Calculator (GUI)

A professional GUI-based BMI calculator built using **Tkinter**.

**Features:**

* BMI calculation
* Category classification
* Input validation
* Reset functionality

---

### 3️⃣ Password Generator (GUI)

A secure password generator with customizable options.

**Features:**

* Adjustable password length
* Option to include numbers
* Option to include symbols
* Copy to clipboard functionality

---

## ▶️ How to Run

1. Install **Python (3.10 or 3.11 recommended)**

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the chat server:

```
python server.py
```

4. Run the client application:

```
python client.py
```

Open multiple client windows to simulate multiple users chatting.
