# 🚀 OASIS INFOBYTE PYTHON INTERNSHIP PROJECTS

This repository contains the projects completed during the 4-week Python Internship Program at Oasis Infobyte.

---

## 📌 Projects Included

### 1️⃣ Voice Assistant
A Python-based voice automation tool capable of:
- Telling current time
- Searching Wikipedia
- Opening websites
- Playing YouTube videos

Technologies:
- SpeechRecognition
- pyttsx3
- pywhatkit
- Wikipedia API

---

### 2️⃣ BMI Calculator (GUI)
A professional GUI-based BMI calculator built using Tkinter.

Features:
- BMI calculation
- Category classification
- Input validation
- Reset functionality

---

### 3️⃣ Password Generator (GUI)
A secure password generator with customizable options.

Features:
- Adjustable length
- Include numbers
- Include symbols
- Copy to clipboard feature

---

## ▶️ How to Run

1. Install Python (3.10 or 3.11 recommended)

2. Install dependencies:
   pip install -r requirements.txt

3. Run main menu:
   python index.py

4. Select project from menu.
