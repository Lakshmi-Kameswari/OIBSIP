# Python Chat Application

A real-time chat application built using Python sockets and Tkinter GUI.

## Features

* Real-time messaging
* Multiple users
* Client-server architecture
* GUI interface
* Emoji support
* Message timestamps

## Technologies Used

* Python
* Socket Programming
* Tkinter
* Threading

## Installation

Clone the repository

```
git clone https://github.com/yourusername/chat-application-python.git
```

Navigate to the folder

```
cd chat-application-python
```

Run the server

```
python server.py
```

Run the client (open multiple terminals)

```
python client.py
```
